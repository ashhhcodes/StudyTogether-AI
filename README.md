# StudyTogether-AI 🚀

An AI-powered collaborative productivity platform designed for students and focused learners.

StudyTogether-AI combines focus management, real-time collaboration, analytics, and AI assistance into a single modern web application.

---

# ✨ Features

## 🔐 Authentication

* Secure user authentication using Supabase
* Login & Signup system
* Protected user-specific data

## ✅ Smart Todo Management

* Create, update, complete, and delete tasks
* Persistent task storage with Supabase
* Priority and category support

## ⏳ Focus & Pomodoro System

* Focus timer with Pomodoro support
* Session tracking
* Study session persistence
* Productivity monitoring

## 📊 Analytics Dashboard

* Study session analytics
* Focus score tracking
* Study consistency heatmap
* Session history
* Dynamic productivity metrics

## 👥 Realtime Study Rooms

* Create collaborative study rooms
* Join study rooms
* Live participant updates using Supabase Realtime
* Dynamic room management

## 🤖 AI Study Assistant

* AI-powered assistant integrated using the OpenRouter API
* Ask study-related doubts
* Generate study plans
* Productivity assistance
* Interactive AI chat interface

## 🎨 Modern UI/UX

* Responsive design
* Animated backgrounds
* Dark modern aesthetic
* Smooth interactive components
* Sidebar navigation system

# ⚙️ Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_OPENROUTER_API_KEY=your_openrouter_api_key
```

# 🚀 Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/ashhhcodes/StudyTogether-AI.git
```

## 2️⃣ Navigate to the project

```bash
cd StudyTogether-AI
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Add environment variables

Create a `.env` file and add the required keys.

## 5️⃣ Run the development server

```bash
npm run dev
```
